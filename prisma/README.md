# 🎓 Campus OS

A modern Full Stack College Management System built using Next.js, TypeScript, Prisma ORM, Supabase PostgreSQL and Tailwind CSS.

---

## 🚀 Features

### 👨‍🎓 Student
- Login & Signup
- Dashboard
- Attendance
- Assignments
- Notes
- Announcements
- Placements

### 👨‍🏫 Faculty
- Faculty Dashboard
- Create Assignments
- Upload Notes
- Post Announcements
- Mark Attendance
- Add Placement Opportunities

### 👨‍💼 Admin
- Admin Dashboard
- Analytics
- Manage Students
- Manage Faculty
- System Overview

---

## 🛠 Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma ORM
- Supabase PostgreSQL
- REST API

---

## 📂 Project Structure

```text
app/

```

## ⚙️ Installation

```bash
git clone <repo-url>

npm install

npx prisma generate

npm run dev
```
