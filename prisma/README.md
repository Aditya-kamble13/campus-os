# 🎓 Campus OS

Campus OS is a modern College Management System developed using **Next.js, Prisma, PostgreSQL (Supabase), Tailwind CSS, and TypeScript**. It provides a single platform for Students, Faculty, and Administrators to manage academic activities efficiently.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Aditya-kamble13/campus-os.git
```

Go to the project folder:

```bash
cd campus-os
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add:

```env
DATABASE_URL=YOUR_DATABASE_URL
DIRECT_URL=YOUR_DIRECT_DATABASE_URL
JWT_SECRET=YOUR_SECRET_KEY
```

Generate the Prisma Client:

```bash
npx prisma generate
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
